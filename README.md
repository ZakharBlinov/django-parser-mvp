# Job Parser - Веб-приложение для сбора вакансий

Веб-приложение для парсинга вакансий с HeadHunter с использованием Django и Django REST Framework.

## Возможности

- Парсинг вакансий с HeadHunter API
- Управление задачами парсинга через админку
- REST API для доступа к данным
- Веб-интерфейс для просмотра вакансий
- Фильтрация и поиск вакансий
